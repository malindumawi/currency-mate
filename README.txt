💱 CurrencyMate - Currency Recognition Mobile App
==================================================

🚀 Project Overview
CurrencyMate is a mobile application designed to assist visually impaired individuals in identifying Sri Lankan currency notes using image recognition techniques. The application is built using Flutter with a focus on accessibility and user-friendly UI/UX design created in Figma.

--------------------------------------------------
🎯 Objectives
--------------------------------------------------
- Enable visually impaired users to identify currency easily
- Provide a simple and accessible mobile interface
- Demonstrate integration of mobile UI with backend detection systems
- Deliver a lightweight and responsive application

--------------------------------------------------
🛠️ Tech Stack
--------------------------------------------------
Frontend:
- Flutter (Cross-platform mobile development)

Backend:
- Flask Server (currency detection API)

Design:
- Figma (UI/UX Design)

Other:
- Dart
- Python
